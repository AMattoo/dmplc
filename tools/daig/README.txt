0. INTRODUCTION

This directory contains the implementation files for the Distributed Algorithm
Intermediate Generator tool, which takes in a MCDA or DAT file and generates
C and/or MADARA files that implement the templated algorithm.

1. PREREQUISITES

Compilation of this tool requires Boost.Spirit and MADARA.

  1.A. MADARA

    http://madara.googlecode.com (Main project site)
    http://code.google.com/p/madara/wiki/Installation (Binary downloads)

  1.B. Boost.Spirit (part of the Boost libraries)

    http://www.boost.org (Main project site)
    http://www.boost.org/users/download/ (Downloads)
